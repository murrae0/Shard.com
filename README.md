# Project Title

## Deployment Guide

### Requirements
- Node.js
- npm

### Setup Instructions
1. Clone the repository.
   ```bash
   git clone https://github.com/murrae0/Shard.com.git
   cd Shard.com
   ```
2. Install dependencies.
   ```bash
   npm install
   ```
3. Setup your Firebase configuration in .env.local file.
4. Run the application.
   ```bash
   npm start
   ```
